Directives

Directives are classes that add new behavior or modify the existing behavior to the elements in the template.

Types of Directives are:
  • Component Directive
  • Structural Directive
  • Attribute Directive
Custom Directives can be created by using angular cli command ng g d directive-name.
Directives use @Directive() decorator.
For Example,
template:`<h1 [appAppHighlight]>Heading </h1>`
@Directive({{ '{' }}
selector: '[appAppHighlight]'
})
export class AppHighlightDirective implements OnInit {{ '{' }}
constructor(private element :ElementRef) {{ '{' }} }
ngOnInit(){{ '{' }}
this.element.nativeElement.style.background = "green"
}
